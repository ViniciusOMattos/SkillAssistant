# SkillAssistant

SkillAssistant é um software simples para automação de teclas e criação de combos personalizados.  
Ele foi desenvolvido para facilitar a execução de sequências de comandos de forma rápida e eficiente.

## 🚀 Funcionalidades

- Criação de **combos de teclas personalizados**
- Definição de **tempo (delay)** entre cada tecla
- Sistema de **perfis**:
  - Criar
  - Salvar
  - Selecionar
- Interface simples e fácil de usar

  
Feito para uso pessoal e aprendizado 💻
